## OLED Display

OLED display compatible con SSD1306, usé Luma.oled para controlarlo [https://github.com/rm-hull/luma.oled](https://github.com/rm-hull/luma.oled) por medio de I2C. En lumatest se puede ver un ejemplo de uso.
